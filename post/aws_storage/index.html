AWS Storage

S3

Simple Storage Service.

S3分类：

  • S3标准
  • S3标准-IA
  • S3单区-IA
  • S3智能分层
  • S3 Glacier
  • S3 Glacire Deep Archive

特点:

  • 无限存储
  • 单个对象达到5Tb
  • 一次写入，多次读取
  • 99%持久性

S3 Glacier

用于存放归档文件，按照访问量收费。

EBS

Elastic Block Store.

https://github.com/kubernetes-sigs/aws-ebs-csi-driver

EBS可以连接到EC2实例，可用区级资源，需要位于同一可用区中才能连接到EC2实例，卷不会自动扩展。

EBS只能被单个EC2实例访问。

特点:

  • 大小高达16Tb
  • EC2实例终止后仍然存在
  • 默认情况下为固态(SSD)
  • 提供IOPS SSD和HDD选项

EFS

https://github.com/kubernetes-sigs/aws-efs-csi-driver

Elastic File System 是linux的NFS服务.

EFS是区域性服务，将数据存储在多个可用区中。可以被上千个EC2实例同时访问。

  • 完全托管的NFS文件系统
  • 为linux负载而设计
  • 支持PB级别数据
  • 可以跨多个区域存储数据

EFS提供两种存储类型(StorageClass): Standard, Infrequent access.

FSx

FSx是windows File Server服务.

Storage Gateway

支持三种类型:

  • Tape gateway
  • Volume gateway
  • File Gateway
