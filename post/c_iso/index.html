C标准库

ISO C89(ANSI C89) -> ISO C95 -> ISO C99 ->

-> ISO C11 -> ISO C17 -> ISO C23

ANSI C 和 ISO C是对通用C语言的接口的定义。

符合这种标准的实现为C语言标准库,也叫libc。

官方网站：

http://www.open-std.org/jtc1/sc22/wg14/www/standards
