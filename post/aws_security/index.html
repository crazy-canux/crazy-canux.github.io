ACM

AWS Certificate Manager.

在route 53创建托管区(hosted zones)之后，可以在ACM申请证书。

申请好了证书，需要在route 53创建记录.

aws acm list-certificates --region eu-west-1

aws-load-balancer-controller可以自动关联ACM的证书和ingress.
