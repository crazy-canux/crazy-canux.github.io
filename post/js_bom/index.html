BOM

Browser Object Model

介绍浏览器对象，不过缺乏规范.

window

window对象是最顶层对象，有6大属性,属性本身也是对象.

window对象的document属性也是对象，document对象有5大属性.

引用当前窗口时，可以省略window对象.

eg:

window.document
