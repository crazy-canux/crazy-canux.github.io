persistentvolumeжІЎжңүnamespace, з”ЁжқҘжҢҮе®ҡе…·дҪ“зҡ„еӯҳеӮЁиө„жәҗгҖӮжңүйқҷжҖҒе’ҢеҠЁжҖҒдёӨз§Қж–№ејҸпјҢжңҖз»ҲйңҖиҰҒз»‘е®ҡеҲ°pvcдёҠгҖӮ

pvзҡ„еӣһж”¶зӯ–з•ҘReclaimPolicy:

  • Retainedдҝқз•ҷ
  • DeletedеҲ йҷӨ

pvзҡ„еҚ·з»‘е®ҡжЁЎејҸvolumeBindingMode:

  • WaitForFirstConsumer

еҚ·жЁЎејҸvolumeMode:

  • Filesystem(й»ҳи®Ө)
  • Block

и®ҝй—®жЁЎејҸaccessMode:

  • RWO: ReadWriteOnce
  • ROX: ReadOnlyMany
  • RWX: ReadWriteMany
  • RWOP: ReadWriteOncePod

еҚ·зҡ„йҳ¶ж®өпјҡ

  • Avaliable
  • Bound
  • Released
  • Failed

dynamic volume provisioning

еҠЁжҖҒpvйңҖиҰҒstorageclass, з”ұStorageClassеҠЁжҖҒзҡ„еҲӣе»әPV, дёҚйңҖиҰҒжүӢеҠЁеҲӣе»әpvпјҢеҸӘйңҖиҰҒеңЁpvcдёӯжҢҮе®ҡstorageclassеҚіеҸҜ.

storageclassжІЎжңүnamespace

kind: StorageClass
apiVersion: storage.k8s.io/v1
metadata:
  name: sourcegraph
labels:
  deploy: sourcegraph
provisioner: kubernetes.io/aws-ebs
parameters:
  type: gp2 
reclaimPolicy: Retain
volumeBindingMode: WaitForFirstConsumer
allowVolumeExpansion: true

apiVersion: v1
kind: PersistentVolumeClaim
metadata:
  name: pgsql
  labels:
    deploy: sourcegraph
    sourcegraph-resource-requires: no-cluster-admin
    app.kubernetes.io/component: pgsql
spec:
  // йҖҡиҝҮstorageClassNameиҮӘеҠЁз»ҷpvcеҲӣе»әpv
  storageClassName: sourcegraph
  accessModes:
  - ReadWriteOnce
  resources:
    requests:
    storage: 200Gi
