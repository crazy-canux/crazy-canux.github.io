WordPress

安装

安装依赖：

1
2
3

$sudo apt install apache2
$sudo apt install mysql-server
$sudo apt install php5

下载wordpress：

1
2

$cd /var/www/html
$git clone https://github.com/WordPress/WordPress.git

配置

添加mysql数据库：

1
2
3
4
5

$mysql -u root -p
mysql> create user 'wordpress'@'localhost' identified by '******';
mysql> create database 'wordpress';
mysql> GRANT ALL PRIVILEGES ON wordpress.* TO "wordpress"@"localhost";
mysql> FLUSH PRIVILEGES;

编辑配置文件：

1

wp-config.php

登陆并安装站点：

1

http://localhost

管理界面：

1

http://localhost/wp-admin

语言

下载相应的语言版本,中文下载中文版的wordpress.

然后将wordpress/wp-content/languages拷贝到英文版的wordpress/wp-content下

语言包路径:

1

wordpress/wp-content/languages

主题

修改wp-config.php安装主题不用输入ftp信息：

1
2
3

define("FS_METHOD", "direct");
define("FS_CHMOD_DIR", 0777);
define("FS_CHMOD_FILE", 0777);

主题安装在：

1

wordpress/wp-content/themes

主题数据导入到:

1

wordpress/wp-content/uploads

插件

插件安装在：

1

wordpress/wp-content/plugins

WordPress Reset

一款可以重置所有数据库数据的插件。

widget-settings-importexport

wordpress-importer

wordpress-move

设置

默认的固定连接:

1

/index.php/%year%/%monthnum%/%day%/%postname%/
