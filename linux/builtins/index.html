Linux内置命令

内置命令在bash/builtins目录中

shell命令分为内置命令和外部命令.

查看一个命令是内置命令还是外部命令：

1
2
3

type -a [command]

提示"[command] is a shell builtin"就表示是内置命令，否则就是外部命令。

查看所有内置命令：

1
2

help
enable -a

查看内置命令的帮助：
